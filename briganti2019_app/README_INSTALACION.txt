APP móvil Briganti / Gandaglia 2019 LNI

Contenido:
- index.html: app móvil/PWA responsive.
- manifest.webmanifest + service-worker.js: permiten instalación como app y uso offline si se sirve desde HTTPS o localhost.
- icon.svg: icono de la app.

Cómo usar en móvil:
1) Subir la carpeta a un servidor web HTTPS, intranet hospitalaria o GitHub Pages/Netlify.
2) Abrir index.html desde el navegador del móvil.
3) En iPhone: Compartir > Añadir a pantalla de inicio.
4) En Android/Chrome: menú ⋮ > Añadir a pantalla de inicio o Instalar app.

También puede abrirse directamente index.html, aunque la instalación/offline puede depender del navegador.
